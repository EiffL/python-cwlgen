# Quick-start guide

## Installation

### Pip

```bash
pip install git+https://gitlab.pasteur.fr/kehillio/pycwl.git#egg=pycwl
```

### Manually

Clone the repository and install pycwl:

```bash
git clone https://gitlab.pasteur.fr/kehillio/pycwl.git
cd pycwl
pip install .
```
